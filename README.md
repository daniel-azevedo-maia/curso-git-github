# Git x GitHub - O Tutorial Prático Definitivo

Bem-vindo ao meu tutorial sobre Git/GitHub!

Construí este material pensando em você, que deseja versionar seus projetos e aprender mais sobre Git de forma prática e eficiente. 

Aqui você encontrará links para os vídeos, bem como recursos adicionais para enriquecer sua aprendizagem.

### Parte 1: Instalação e primeiros comandos
Na primeira parte do tutorial, exploramos os fundamentos do Git. Você aprenderá sobre configuração inicial, criação de repositórios, e os comandos básicos para começar seu projeto.
